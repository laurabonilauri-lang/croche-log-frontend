# 🎨 Ateliê de Crochê - Front-end (SPA)

Interface web para o gerenciamento do **Ateliê de Crochê**. Uma Single Page Application (SPA) responsiva que permite a artesã visualizar seu dashboard de produtividade e gerenciar suas peças em tempo real.

---

## 💻 Funcionalidades da Interface

- **Dashboard**: Exibição dinâmica do total de projetos e peças finalizadas.
- **Gestão de Cards**: Visualização limpa de cada projeto com ações rápidas.
- **Integração Assíncrona**: Comunicação via `Fetch API` com o Back-end, permitindo atualizações sem recarregar a página.
- **UX/UI**: Design focado na facilidade de uso, utilizando Flexbox para responsividade.

## 🛠️ Tecnologias Utilizadas

- **HTML5** & **CSS3** (Variáveis CSS e Flexbox).
- **JavaScript Vanilla**: Lógica de integração e manipulação do DOM.

## 🚀 Como Executar

1. Certifique-se de que a [API do projeto](https://github.com/laurabonilauri-lang/croche-log-api) esteja rodando localmente na porta 5000.
2. Navegue até a pasta `frontend` em seu computador.
3. Localize o arquivo **`index.html`** e abra-o diretamente com o navegador de sua preferência (ex: Chrome, Edge ou Firefox).
4. **Não é necessário** utilizar extensões (como Live Server), servidores locais ou configurações adicionais.
---

## 🔗 Integração
Este Front-end depende da API desenvolvida em Python/Flask para funcionar corretamente.
